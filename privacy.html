Datenschutzrichtlinie

SchreibAssistent Pro – Microsoft Word Add-in

Letzte Aktualisierung: 23. Februar 2026

1. Verantwortlicher

SchreibAssistent Pro ist ein Open-Source-Projekt. Den Quellcode finden Sie unter github.com/sucruf111/SchreibAssistentPro.

2. Welche Daten werden verarbeitet?

SchreibAssistent Pro verarbeitet ausschließlich den Text, den Sie aktiv zur Analyse an das Add-in senden (z. B. durch Klicken auf "Korrektur starten" oder "Vorschläge generieren"). Es werden keine Daten automatisch oder im Hintergrund gesammelt.

  • Dokumenttext: Der von Ihnen ausgewählte oder zur Analyse freigegebene Text wird an die OpenAI-API gesendet, um Korrekturen, Lektorat, Stilanalyse oder Vorschläge zu erstellen.
  • OAuth-Token: Wenn Sie sich mit Ihrem OpenAI-Konto anmelden, wird ein Zugriffstoken lokal in den Office-Roaming-Einstellungen gespeichert.

3. Welche Daten werden NICHT erhoben?

  • Keine personenbezogenen Daten (Name, E-Mail, IP-Adresse)
  • Keine Nutzungsstatistiken oder Telemetriedaten
  • Kein Tracking, keine Cookies, keine Analyse-Tools
  • Keine Speicherung Ihrer Texte auf unseren Servern

4. Drittanbieter-Dienste

Ihr Text wird zur Verarbeitung direkt an die OpenAI-API gesendet. Die Verarbeitung erfolgt gemäß den Datenschutzrichtlinien von OpenAI. Es werden keine weiteren Drittanbieter-Dienste genutzt.

5. Datenspeicherung

Alle Daten werden ausschließlich lokal in Ihrem Browser bzw. in den Office-Roaming-Einstellungen gespeichert:

  • Stilprofile (über Office.context.roamingSettings)
  • OAuth-Token (über Office.context.roamingSettings)
  • Einstellungen (Korrekturmodus, Fachgebiet, Zitierstil)

Es werden keine Daten an eigene Server gesendet oder dort gespeichert.

6. Ihre Rechte

Da keine personenbezogenen Daten erhoben oder gespeichert werden, sind Auskunfts-, Löschungs- und Berichtigungsanfragen in der Regel gegenstandslos. Sie können jederzeit das OAuth-Token durch Abmelden entfernen und das Add-in deinstallieren.

7. Open Source

SchreibAssistent Pro ist vollständig quelloffen. Sie können den gesamten Quellcode einsehen und prüfen, welche Daten wie verarbeitet werden.

8. Kontakt

Bei Fragen zum Datenschutz erstellen Sie bitte ein Issue auf GitHub.

Privacy Policy (English)

1. Data Controller

SchreibAssistent Pro is an open-source project. Source code is available at github.com/sucruf111/SchreibAssistentPro.

2. What data is processed?

SchreibAssistent Pro only processes text you actively submit for analysis (e.g., by clicking "Start correction" or "Generate suggestions"). No data is collected automatically.

  • Document text: Text you select or submit is sent to the OpenAI API for grammar checking, proofreading, style analysis, or suggestions.
  • OAuth token: When you sign in with your OpenAI account, an access token is stored locally in Office roaming settings.

3. What data is NOT collected?

  • No personal data (name, email, IP address)
  • No usage statistics or telemetry
  • No tracking, cookies, or analytics
  • No server-side storage of your texts

4. Third-party services

Your text is sent directly to the OpenAI API for processing, subject to OpenAI's privacy policy. No other third-party services are used.

5. Data storage

All data is stored locally in your browser or Office roaming settings. No data is sent to or stored on our servers.

6. Contact

For privacy questions, please open an issue on GitHub.
